# Panduan Penggunaan Aplikasi

Selamat datang di FitScript AI.

## 1. Mulai menggunakan aplikasi

- Masuk dengan Google, Apple, atau lanjut sebagai tamu.
- Pilih bahasa aplikasi dari menu yang tersedia.

## 2. Upload dokumen hasil lab

- Gunakan kamera, galeri, atau dokumen PDF.
- Pastikan file yang dipilih jelas dan dapat dibaca.

## 3. Analisis dan simpan hasil

- Tekan tombol analisis untuk memproses hasil lab.
- Simpan hasil analisis untuk melihat riwayat dan perkembangan.

## 4. Akun dan sinkronisasi

- Hubungkan akun Google/Apple untuk sinkronisasi data lintas perangkat.
- Mode tamu menyimpan data secara lokal pada perangkat.

## 5. Catatan penting

FitScript AI adalah alat bantu edukasi dan informasi. Hasil aplikasi bukan pengganti diagnosis atau saran medis profesional.

## 6. Kontak dukungan

Jika terjadi kendala penggunaan, silakan hubungi tim pengembang melalui kanal dukungan resmi aplikasi.
