By using FitScript AI, you agree to this Privacy Policy.

## 1. App Identity

- **App Name:** FitScript AI: Cek Hasil Lab
- **Summary:** Pahami hasil lab medis, cek darah & kesehatan dengan penjelasan AI yang mudah.
- **Core Service:** Educational AI assistance for understanding lab test results.

## 2. Data We Collect

Depending on how you use the app, we may process:

1. **Account Data**
   - Anonymous user ID (automatic guest session)
   - Linked identity data (Google or Apple), such as email and display name

2. **Health & Lab Data**
   - Images/documents you upload (lab report photos)
   - Extracted medical values and interpreted results
   - Analysis history and trends

3. **Usage & Device Data**
   - Basic analytics events
   - App performance and error logs
   - Device/platform metadata required for app functionality

4. **Subscription / Plan Metadata**
   - Plan status (for example: free/premium)
   - Feature usage counters and restore-purchase related metadata

## 3. How We Use Your Data

We use data to:

- Analyze lab results and present easier explanations
- Send the lab report file you upload (image or PDF), including any personal health data it contains, to a third-party AI service (Google Cloud Vertex AI / Gemini) for analysis and insight generation. This is done only with your explicit consent, and you can revoke it in the app settings.
- Provide personalized lifestyle suggestions (activity and nutrition guidance)
- Store history so you can monitor trends over time
- Sync your data across devices when you link an account
- Improve app quality, reliability, and security
- Provide support and resolve technical issues
